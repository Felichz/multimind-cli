**Format–Evidence Coupling** — The confabulation risk of a scoreboard/table is independent of whether each cell is factually correct. A formatted summary without the corresponding raw output beneath it is always a confabulation signal, because the format is the agent's own editorial assertion. When an agent reports task completion, it must present raw tool outputs first (verbatim) and only then a structured interpretation. The structured format must be directly traceable to line items in the raw output. If an agent cannot cite the exact raw output lines that support each row of a table, the table must not be presented.
